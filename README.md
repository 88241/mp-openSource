
## 项目功能

**AsyncAwait**

* 在小程序里使用ES7的async-await

**GlobalReload**

* 处理服务器返回失败后，加载失败页面以及点击重新加载，加载网络请求失败页

**ShareImage**

* 用于绘制图片，分享给好友或者是发送到朋友圈

**NavigationBar**

* 自定义导航栏，可用于全局跟单页面

**DropdownView**

* 自定义下拉三级展示菜单
 
**Components(一些常用组件)**

* topHint 当输入账号密码错误等情况下的错误提示
* toastImageText 自定义弹窗

**DatePicker**

* 预定酒店日期选择器
